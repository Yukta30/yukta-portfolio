# Yukta Vajpayee — Portfolio (Dark Theme)

Personalized starter matching your UX and content, using a black/gray + purple accent theme.

## Structure
- `index.html` — content and sections
- `styles.css` — dark theme + components
- `script.js` — nav highlight + skill-bar animation
- `assets/icons` — purple-stroked SVG icons
- `assets/images` — placeholders (replace with real images)
- `resume.pdf` — your attached resume

## Local Dev
Open `index.html` or serve with:
```
python -m http.server 8000
```

## Deploy to GitHub Pages
1. Create a repo (e.g. `yukta-username.github.io`).
2. Upload all files.
3. Visit `https://<your-username>.github.io`.

Enjoy tweaking colors in `:root` (see `--brand` for purple). To try a gray accent, change `--brand`/`--brand-2` to slate/neutral values.
